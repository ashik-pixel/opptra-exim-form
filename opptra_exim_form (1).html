Setup Guide — Connect your form

This setup takes about 5 minutes. Every form submission will add a new row to your Google Sheet and send you an email notification automatically.
1
Create a Google Sheet

Go to sheets.google.com → create a new blank sheet. Name it "Opptra EXIM Submissions". Leave it open.

2
Open Apps Script

In your Sheet, click Extensions → Apps Script. A new tab opens with a code editor. Delete all existing code.

3
Paste this script

Copy the full script below and paste it into the editor:

const SHEET_NAME = "Sheet1";
const NOTIFY_EMAIL = "your@email.com"; // ← Change this

function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const ss = SpreadsheetApp.getActiveSpreadsheet();
    const sheet = ss.getSheetByName(SHEET_NAME);

    // Build header row on first submission
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(Object.keys(data));
    }
    sheet.appendRow(Object.values(data));

    // Email notification
    const subject = "New EXIM Request: " + data.ref_number;
    let body = "New shipment request received from Opptra EXIM form.\n\n";
    for (const [k, v] of Object.entries(data)) {
      body += k.replace(/_/g," ") + ": " + v + "\n";
    }
    MailApp.sendEmail(NOTIFY_EMAIL, subject, body);

    return ContentService
      .createTextOutput(JSON.stringify({status:"ok"}))
      .setMimeType(ContentService.MimeType.JSON);

  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({status:"error", message: err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  return ContentService
    .createTextOutput(JSON.stringify({status:"ready"}))
    .setMimeType(ContentService.MimeType.JSON);
}
⚠ Change your@email.com on line 2 to your actual email before deploying.
4
Deploy as a Web App

Click Deploy → New Deployment. Set type to Web app. Set "Execute as" to Me. Set "Who has access" to Anyone. Click Deploy.

5
Copy the Web App URL

After deploying, Google shows you a URL like https://script.google.com/macros/s/.../exec. Copy it and paste it into the setup banner at the top of the form.

✓ Done! Every submission now saves a row in your Google Sheet and sends you an email with all the details.
Formspree is a free service that emails you every form submission. Free plan allows 50 submissions/month. No server setup needed.
1
Create a free Formspree account

Go to formspree.io and sign up with your email.

2
Create a new form

Click + New Form. Give it a name like "Opptra EXIM". Enter the email address where you want submissions sent.

3
Copy the endpoint URL

Formspree gives you a URL like https://formspree.io/f/xxxxxxxx. Copy it.

4
Paste into the setup banner

Paste the URL into the Email (Formspree) tab in the orange setup banner at the top of the form and click Save.

✓ Done! Every submission will be emailed to you instantly with all form fields.
